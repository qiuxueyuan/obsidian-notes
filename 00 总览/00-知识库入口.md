## 🎯 使用指南

### 快速开始

1. **日常记录** → 在 `[[05 学习笔记]]` 中创建新笔记
2. **项目记录** → 在 `[[03 项目实战]]` 中记录踩坑与方案
3. **刷题打卡** → 在 `[[04 面试刷题]]` 中记录算法题
4. **查找资料** → 使用 `Cmd/Ctrl + O` 快速搜索

### ⚙️ 设置首页（重要）

在 Obsidian 中设置此笔记为打开首页：

1. 打开 **设置** → **文件与链接**
2. 找到 **打开此笔记** 选项
3. 选择 `00-知识库入口`

### 📦 插件安装

**Dataview 插件** - 让总览页面的统计和查询生效：

1. 打开 **设置** → **第三方插件** → **浏览**
2. 搜索 `Dataview`
3. 点击 **安装** → **启用**
4. 重启 Obsidian（如需要）
