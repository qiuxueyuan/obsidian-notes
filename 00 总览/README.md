# 00 总览

> ⚠️ **在此文件夹创建文件前，请先读取本 README 并遵循命名规范。**

---

## 📌 目录用途

知识库的**入口和导航中心**，用于：
- 提供知识库整体地图和快速导航
- 展示待办事项和任务追踪
- 显示最近更新的笔记
- 提供知识库统计数据

## 📁 适用内容

| 内容类型 | 说明 | 示例 |
|----------|------|------|
| 入口导航 | 知识库首页、技能地图 | `00-知识库入口.md` |
| 待办清单 | 任务追踪、进度管理 | `待办-项目复盘.md` |
| 索引文件 | 各主题的索引导航 | `索引 - 技能地图.md` |
| 统计数据 | Dataview 查询结果 | `统计 - 学习笔记.md` |

## 📝 命名规范

**格式**：`MMDD + 文件名.md`

- **MMDD**：4 位数字，表示创建日期（月 + 日）
- **文件名**：简洁描述内容，使用中文或英文

### 示例

```
00 总览/
├── 0301-知识库入口.md        # 3 月 1 日创建
├── 0302-本周待办.md          # 3 月 2 日创建
├── 0302-技能地图.md          # 3 月 2 日创建
└── 0302-月度复盘模板.md       # 3 月 2 日创建
```

### 特殊文件

- `00-知识库入口.md` - 固定为知识库首页（使用 00 作为特殊标识）
- `99-归档索引.md` - 固定为归档文件索引
