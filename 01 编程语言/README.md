## 📌 目录用途

**各编程语言的核心知识点**，用于：
- 记录语言语法、特性和最佳实践
- 整理语言相关的技术深度内容
- 积累代码示例和常见用法
- 形成系统化的语言知识体系

## 📁 适用内容

| 内容类型 | 说明 | 示例 |
|----------|------|------|
| 语言核心 | 语法、类型系统、特性 | `030101-Python 装饰器详解.md` |
| 标准库 | 内置模块、常用 API | `030102-Python itertools 详解.md` |

| 对比总结 | 多语言对比分析 | `030301-Python vs Go 并发对比.md` |

## 📝 命名规范

**格式**：`MMDD + 序号 + 文件名.md`

- **MMDD**：4 位数字，表示创建日期（月 + 日）
- **序号**：2 位数字，表示当日创建的顺序（01, 02, 03...）
- **文件名**：`语言名 + 主题` 或 `主题名`

### 示例

```
01 编程语言/
├── 022501-Python 装饰器详解.md       # 2 月 25 日创建的第1个文件
├── 022502-Go 协程与 Channel.md       # 2 月 25 日创建的第2个文件
├── 022601-Java 并发包核心类.md       # 2 月 26 日创建的第1个文件
├── 030101-TypeScript 泛型进阶.md     # 3 月 1 日创建的第1个文件
├── 030102-Rust 所有权系统.md         # 3 月 1 日创建的第2个文件
└── 030201-鸿蒙 ArkTS 基础语法.md     # 3 月 2 日创建的第1个文件
```
