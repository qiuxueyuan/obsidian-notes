# 04 面试刷题

> ⚠️ **在此文件夹创建文件前，请先读取本 README 并遵循命名规范。**

---

## 📌 目录用途

**面试准备与算法练习**，用于：
- 记录算法题解题思路和代码
- 整理面试考点和八股文
- 追踪刷题进度和正确率
- 积累系统设计案例

## 📁 适用内容

| 内容类型 | 说明 | 示例 |
|----------|------|------|
| 算法题 | LeetCode、剑指 Offer 等 | `030101-LeetCode-两数之和.md` |
| 八股文 | 基础知识、概念理解 | `030102-HTTP 与 HTTPS 区别.md` |
| 系统设计 | 架构设计、场景题 | `030201-设计短链系统.md` |
| 面经 | 面试真题、经历记录 | `030202-字节后端一面面经.md` |
| 进度追踪 | 刷题计划、统计 | `030301-刷题进度 -2026Q1.md` |

## 📝 命名规范

**格式**：`MMDD + 序号 + 文件名.md`

- **MMDD**：4 位数字，表示创建日期（月 + 日）
- **序号**：2 位数字，表示当日创建的顺序（01, 02, 03...）
- **文件名**：`题目名` 或 `考点名` 或 `公司 + 轮次 + 面经`

### 示例

```
04 面试刷题/
├── 022501-LeetCode-1. 两数之和.md    # 2 月 25 日创建的第1个文件
├── 022502-剑指 Offer-链表反转.md     # 2 月 25 日创建的第2个文件
├── 022601-HTTP 三次握手详解.md       # 2 月 26 日创建的第1个文件
├── 030101-设计短链系统.md            # 3 月 1 日创建的第1个文件
├── 030102-字节后端一面面经.md        # 3 月 1 日创建的第2个文件
└── 030201-刷题进度 -2026Q1.md        # 3 月 2 日创建的第1个文件
```
