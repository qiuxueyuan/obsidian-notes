# 05 学习笔记

> ⚠️ **在此文件夹创建文件前，请先读取本 README 并遵循命名规范。**

---

## 📌 目录用途

**学习资料摘录**，用于：
- 记录课程、文档、博客的学习内容
- 整理技术文章的核心要点
- 积累知识卡片和心得
- 形成持续学习的知识库

## 📁 适用内容

| 内容类型 | 说明 | 示例 |
|----------|------|------|
| 课程笔记 | 在线课程、培训记录 | `极客时间-Go 进阶课.md` |
| 文档摘录 | 官方文档、技术文档 | `React 官方文档摘录.md` |
| 博客摘要 | 技术博客、文章心得 | `某大牛博客摘要.md` |
| 读书心得 | 技术书籍读后感 | 《代码整洁之道》心得.md |
| 技术分享 | 内部分享、会议记录 | `技术分享 - 微服务架构.md` |

## 📝 命名规范

**格式**：`MMDD + 文件名.md`

- **MMDD**：4 位数字，表示创建日期（月 + 日）
- **文件名**：`来源 + 主题` 或 `主题名`

### 示例

```
05 学习笔记/
├── 0225-极客时间-Go 进阶课 01.md   # 2 月 25 日创建
├── 0226-React 官方文档摘录.md      # 2 月 26 日创建
├── 0228-深入理解计算机系统心得.md  # 2 月 28 日创建
├── 0301-技术分享 - 微服务架构.md   # 3 月 1 日创建
├── 0301-Obsidian 同步教程.md       # 3 月 1 日创建
└── 0302-Kubernetes 权威指南笔记.md # 3 月 2 日创建
```

### 子目录建议（可选）

```
05 学习笔记/
├── 课程笔记/         # 在线课程、培训
├── 文档摘录/         # 官方文档、技术文档
├── 博客摘要/         # 技术博客、文章
├── 读书心得/         # 技术书籍
└── 技术分享/         # 内部分享、会议
```

## 🏷️ 标签建议

- `#课程` `#文档` `#博客` `#读书` `#技术分享`
- `#进行中` `#已完成` （学习状态）
- `#python` `#go` `#react` （技术标签）
