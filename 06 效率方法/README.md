# 06 效率方法

> ⚠️ **在此文件夹创建文件前，请先读取本 README 并遵循命名规范。**

---

## 📌 目录用途

**效率提升与工作方法**，用于：
- 记录编程效率和工具使用技巧
- 整理学习方法和知识管理
- 优化工作流程和时间管理
- 积累可复用的模板和脚本

## 📁 适用内容

| 内容类型 | 说明 | 示例 |
|----------|------|------|
| 编程效率 | IDE 配置、快捷键、Snippets | `VSCode 高效配置.md` |
| 工具技巧 | 命令行、自动化脚本 | `Zsh 配置与插件.md` |
| 学习方法 | 费曼技巧、刻意练习 | `费曼学习法实践.md` |
| 工作流程 | GTD、番茄工作法 | `GTD 实践指南.md` |
| 模板库 | 周报、OKR、复盘模板 | `周报模板.md` |

## 📝 命名规范

**格式**：`MMDD + 文件名.md`

- **MMDD**：4 位数字，表示创建日期（月 + 日）
- **文件名**：`主题名` 或 `工具名 + 技巧`

### 示例

```
06 效率方法/
├── 0225-VSCode 高效配置.md        # 2 月 25 日创建
├── 0226-Zsh 配置与插件.md         # 2 月 26 日创建
├── 0228-费曼学习法实践.md         # 2 月 28 日创建
├── 0301-GTD 实践指南.md           # 3 月 1 日创建
├── 0301-周报模板.md              # 3 月 1 日创建
└── 0302-自动化脚本集合.md         # 3 月 2 日创建
```

### 子目录建议（可选）

```
06 效率方法/
├── 编程效率/         # IDE、工具配置
├── 学习方法/         # 学习技巧、知识管理
├── 工作流程/         # GTD、时间管理
├── 模板库/           # 周报、OKR、复盘
└── 自动化脚本/       # 效率脚本
```
