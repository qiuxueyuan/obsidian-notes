# 99 资源附件

> ⚠️ **在此文件夹创建文件前，请先读取本 README 并遵循命名规范。**

---

## 📌 目录用途

**资源库和附件存储**，用于：
- 存储图片、截图、图表等视觉资源
- 积累代码片段和配置模板
- 收集电子书、文档、白皮书
- 整理外部资源和参考链接

## 📁 适用内容

| 内容类型 | 说明 | 示例 |
|----------|------|------|
| 图片资源 | 截图、流程图、架构图 | `architecture.png` |
| 代码片段 | 常用工具函数、模板 | `utils.py` |
| 文档资源 | PDF、电子书、白皮书 | `《代码整洁之道》.pdf` |
| 外部资源 | 网站收藏、视频链接 | `常用参考网站.md` |
| 思维导图 | XMind、MindNode 文件 | `知识体系.xmind` |

## 📝 命名规范

**格式**：`MMDD + 文件名。扩展名`

- **MMDD**：4 位数字，表示创建日期（月 + 日）
- **文件名**：简洁描述内容，使用英文或拼音（方便引用）

### 示例

```
99 资源附件/
├── 0225-architecture.png          # 2 月 25 日创建
├── 0226-utils.py                 # 2 月 26 日创建
├── 0228-docker-compose.yml       # 2 月 28 日创建
├── 0301-知识体系.xmind            # 3 月 1 日创建
├── 0301-常用参考网站.md          # 3 月 1 日创建
└── 0302-snippet-collection.md    # 3 月 2 日创建
```

### 子目录建议（可选）

```
99 资源附件/
├── 图片/             # PNG、JPG、GIF、WebP
├── 代码/             # 代码片段、脚本
├── 配置/             # 配置文件模板
├── 文档/             # PDF、电子书
└── 外部资源/         # 链接集合
```

## 🏷️ 标签建议

- `#图片` `#代码` `#配置` `#文档` `#外部资源`
- `#截图` `#架构图` `#模板` `#电子书`
